# Tiktok-Downloader
> 一款 抖音/tiktok 无水印下载插件

### 注：目前版本只支持进入详情页下载!!!

### 安装

在`release`页面下载crx文件，如果不能使用，直接下载zip文件解压后，然后直接把整个文件夹拖到浏览器里（注意要打开开发者模式）
